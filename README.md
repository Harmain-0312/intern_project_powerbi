# Power BI Dashboard Project

This repository contains a comprehensive Power BI dashboard project developed to analyze and visualize job market data. The dashboard provides meaningful insights into job trends, salaries, company characteristics, and hiring patterns across different countries, industries, and job roles.

## Project Overview

The objective of this project is to transform raw job-related data into interactive and visually appealing dashboards that support data-driven decision-making. Using Power BI, the dataset was cleaned, modeled, and analyzed to uncover key insights about the global job market.

## Key Features

* Interactive and dynamic dashboards
* Data cleaning and transformation using Power Query
* Data modeling with relationships between tables
* Custom DAX measures and calculated columns
* Advanced data visualizations for business insights
* User-friendly navigation and filtering options

## Dashboard Insights

* Job distribution by country, company, and industry
* Salary analysis across job roles and experience levels
* Employment type and work setting trends
* Top hiring companies and most in-demand job roles
* Geographic analysis of job opportunities
* Company size and workforce distribution

## Tools and Technologies Used

* **Power BI Desktop**

* **Microsoft Excel** (for initial data preparation)
